🔍 实验一：光轴-测距轴一致性验证

核心问题：超声波15°波束与LED 60-120°光斑的匹配度

实验步骤

  1. 搭建测试台：LED 灯固定于龙门架，距桌面高度可调（20-80cm）；超声波模块安装于 LED 基板几何中心，发射面向下；桌面铺设坐标纸（10cm×10cm 网格）
  2. 光斑测绘（关闭环境光）：高度设定：30cm、50cm、70cm 三档；每档用照度计测量网格交点处的 PPFD，绘制等值线图；标记光斑中心（最强点）与几何中心偏移量
  3. 超声波测距验证：在光斑范围内放置硬纸板于不同位置（中心点、边缘点、倾斜 30°），记录超声波读数 vs 激光测距真值
  4. 多目标测试（关键）：同时放置 3 张硬纸板于不同高度，观察超声波返回值（是最近？最远？还是平均值？）

预期结论（用于算法修正）：

  • • 若超声始终返回最近距离，算法需采用 d_max 补偿策略（防止高株灼伤）
  • • 若光斑中心与几何中心偏移 > 5cm，需机械调整超声安装位置

📊 实验二：距离-PPFD 标定与叠加计算

核心问题：PPFD_total = PPFD_led + PPFD_ambient 是否成立？

实验步骤

  1. 基础标定（暗室）：调节灯高：20、30、40、50、60、80、100cm；每高度测量中心点 PPFD（LED 100% 亮度）；建立距离-PPFD 基础曲线（反平方律验证）
  2. 亮度线性度测试（固定高度 50cm）：LED 亮度：10%、25%、50%、75%、100%；测量各档 PPFD，验证 PWM 调光是否线性
  3. 环境光叠加验证（关键实验）：
    • 组 A（静态环境光）：侧向放置环境光源（模拟窗户），调至 3 个档位；测量无 LED 时的基础 PPFD_amb；开启 LED（50% 亮度），测量 PPFD_total；验证：PPFD_total = PPFD_amb + PPFD_led（单独测量值）
    • 组 B（动态闭环模拟）：设定目标 PPFD_target = 300 μmol；手动调节环境光变化，用实验数据计算所需 LED 亮度：B% = (300 - PPFD_amb) / PPFD_led(100%) × 100%

⚙️ 数据分析与算法修正

1. LUT 修正

若实测 PPFD_实测 与理论 偏差 > 15%，采用分段线性插值替代公式：

# 基于实验数据的 LUT（示例）
PPFD_TABLE = {
    20: 820,   # 实测值
    30: 480,   # 非理论 365，因透镜聚光
    40: 310,
    50: 215,
    60: 155,
    80: 85,
    100: 55
}

2. 环境光系数修正

若发现 PPFD_total ≠ PPFD_led + PPFD_amb，引入修正系数 α：

PPFD_total = PPFD_led + α × PPFD_amb

通过实验二数据拟合 α（通常在 0.9-1.1 之间）

3. 超声波补偿策略

若实验一发现多目标时超声返回最近距离（d_nearest），而光斑边缘实际距离为 d_edge，算法应保守取：

d_safe = min(d_edge, d_nearest + margin)

防止边缘植物光照不足

📝 实验报告模板

摘要

验证超声波中心测距结合环境光补偿的控制策略，实测目标 PPFD 达成率 92%，误差主要来源于环境光空间不均匀性。

关键发现

  1. 1. 光斑中心与几何中心偏移 2cm，在可接受范围
  2. 2. 距离-PPFD 关系在 30-80cm 范围接近 1/d（非平方反比，因非点光源）
  3. 3. 环境光叠加误差 < 8%，满足 DLI 日累积控制精度要求

算法优化建议

  • • 增加温度补偿（LED 光效随温升降低，约 -0.3%/℃）
  • • 增加时间积分（10 分钟调节周期平滑波动）
