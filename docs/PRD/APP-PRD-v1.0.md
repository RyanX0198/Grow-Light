# AeroGrow Pro - 智慧植物养护 APP 项目规范文档

## 1. 项目概述

**产品名称**：AeroGrow Pro  
**产品定位**：智能植物照明灯控制与植物养护管理 APP  
**目标平台**：移动端 Web（PWA 适配）  
**核心技术**：HTML5 + Tailwind CSS + Canvas API + 原生 JavaScript  

### 1.1 核心功能
- **智能光环境控制**：PPFD 实时监测与自动补光
- **AR 空间光场**：可视化光照强度分布热力图
- **植物健康管理**：多植物 DLI 追踪与位置优化建议
- **订阅服务**：三级会员体系（Seed/萌芽版/Bloom）

## 3. 功能模块详细说明

### 3.1 Onboarding 引导流程
**触发条件**：首次打开 APP 或点击植物页右上角 "+"  
**流程设计**：
1. **Step 1 - 欢迎页**：品牌展示 + 开始配置按钮
2. **Step 2 - 扫描动画**：雷达扫描效果（2秒自动跳转），显示检测到 3 株植物
3. **Step 3 - 植物选择**：选择主栽植物（龟背竹/绿萝），影响后续 DLI 目标值

**技术实现**：
- 进度条动画（`width` 过渡）
- 使用 `hidden` 类控制步骤切换
- 选择主栽植物后更新 `plant-count` 文本

### 3.2 植物页（首页）功能

#### 3.2.1 DLI 统计卡片
显示四项核心指标：
- **目标 DLI**：14.2 mol/m²/d（基于主栽植物计算）
- **实际 DLI**：10.8 mol/m²/d（今日累计）
- **环境光**：310 μmol/m²s（当前自然光）
- **补光功率**：150 μmol/m²s（当前补光灯输出）

#### 3.2.2 PPFD 时间曲线图（Canvas）
**图表类型**：面积图 + 折线图  
**数据维度**：24 小时光照曲线

**视觉元素**：
- **自然光区域**：深绿色渐变（6:00-19:00 正弦曲线）
- **补光区域**：亮绿色渐变（自动补偿缺口）
- **总光照曲线**：发光描边效果（`shadowBlur`）
- **目标线**：虚线显示（DLI 14.2 对应 0.85 高度）
- **当前时间指示器**：NOW 标签 + 垂直指示线（14:30 位置）

**交互功能**：
- 鼠标悬停显示 Tooltip（时间/自然光/补光/总计数值）

**算法逻辑**：
```javascript
// 自然光模型：正弦曲线模拟日出日落
natural = sin((hour - 5) / 14 * π) * 0.55

// 自动补光：当自然光 < 目标值时启动
boost = (target - natural) * factor
```

#### 3.2.3 植物列表
**数据结构**（示例）：
```javascript
{
  name: "龟背竹",
  icon: "🌿",
  targetDLI: 12,
  currentDLI: 9.1,
  status: "主栽", // 标签显示
  borderColor: "border-green-500"
}
```

**状态标签**：
- **适应良好**（绿色）：当前 DLI ≥ 目标值 70%
- **边缘光弱**（黄色）：当前 DLI < 目标值 70%，建议调整位置

### 3.3 灯光页功能

#### 3.3.1 AR 空间热力图
**技术实现**：Canvas 绘制同心圆渐变 + HTML 叠加植物标记

**光照分区**（5 个同心圆）：
1. **>400 μmol/m²s**（红色）：高光充足区
2. **300-400**（黄色）：良好生长区  
3. **200-300**（绿色）：中等光照区
4. **100-200**（蓝色）：弱光区
5. **<100**（深蓝）：光照不足区

**AR 植物标记**：
- 浮动动画（`float` 3s 循环）
- 脉冲光环（`pulse-ring`）
- 实时数值显示（随滑块调节动态更新）
- 点击显示植物详情 Toast

**交互功能**：
- 鼠标悬停显示坐标 + 该点 PPFD 值 + 状态评估
- 滑块调节中心亮度（100-800），热力图实时重绘

#### 3.3.2 光强调节
- **滑块范围**：100 - 800 μmol/m²s
- **默认值**：480
- **联动效果**：调节时同步更新 AR 标记数值和中心峰值显示

#### 3.3.3 光谱控制
| 参数 | 范围 | 默认值 | 说明 |
|------|------|--------|------|
| 450nm 蓝光 | 0-100% | 65% | 促进叶绿素合成 |
| CCT 色温 | 2700-6500K | 4000K | 冷暖光调节 |
| 730nm 远红光 | 0-4h/夜 | 2h | 光形态建成 |

## 7. 技术实现要点

### 7.1 Canvas 图表
**PPFD 曲线图**：
- 使用 `quadraticCurveTo` 绘制平滑曲线
- 设备像素比适配（`window.devicePixelRatio`）
- 响应式重绘（resize 事件监听）

**AR 热力图**：
- `createRadialGradient` 创建径向渐变
- 从外向内绘制（正确叠加顺序）
- 等值线使用 `arc` 绘制白色描边

### 7.2 响应式适配
- **移动端优先**：viewport 设置 `maximum-scale=1.0, user-scalable=no`
- **安全区域**：底部导航使用 `fixed` 定位，主内容区 `pb-24` 避让
- **Canvas 缩放**：根据容器尺寸动态调整 canvas 大小

### 7.3 性能优化
- 使用 `transform` 和 `opacity` 实现动画（GPU 加速）
- Canvas 离屏渲染（避免频繁重绘）
- 事件委托（聊天消息动态添加）
