# AG (AeroGrow Pro) 项目背景资料

## 项目基本信息
- **项目代号**：AG
- **项目全称**：AeroGrow Pro / Wyze Grow Light
- **核心理念**：全场景 AI 智慧生态补光系统
- **产品口号**：植物光能的"智慧喂养者" + "自然光专家"

## 目标市场
- **地区**：北美室内园艺市场
- **定价**：$69.99（BOM控制在$38-42）
- **策略**：1/3高端价格，3×低端体验

## 五大核心价值

| 价值 | 实现方案 |
|:---|:---|
| **好光源** | Ra≥98、Samsung LM301H EVO、730nm远红光、非对称偏光透镜 |
| **零门槛智慧** | AI视觉识别植物品种 + ToF测距 + DLI闭环自动补光 |
| **量化生长** | 毫米级高度追踪 + 延时影像合成 |
| **健康哨兵** | 边缘AI识别病虫害（红蜘蛛/白粉病/脱水）|
| **全场景适配** | 吊装/立柱/贴壁三种形态 |

## 核心硬件规格（成本优化版）

| 组件 | 规格 |
|:---|:---|
| **主LED阵列** | Samsung LM301H EVO × 30-40颗，Ra95+，2700K-6500K动态色温 |
| **远红光通道** | 730nm远红光 × 4颗，Phytochrome Switch技术 |
| **光学透镜** | 非对称偏光PMMA透镜，45°定向投光 |
| **视觉传感器** | 1080p工业级广角摄像头 |
| **测距传感器** | 单点激光ToF，±1cm精度 |
| **环境光感应** | 集成式数字光敏电阻，支持DLI闭环管理 |
| **主控芯片** | ESP32-S3 (WiFi+BLE)，8MB Flash |
| **电源驱动** | Mean Well LDD-700HW，1%-100%深度调光 |
| **散热方案** | 一体化铝挤型材散热器，被动式散热，无风扇静音 |

## 核心差异化技术

| 技术 | 效果 |
|:---|:---|
| **730nm远红光** | Phytochrome Switch，诱导植物深度休眠，缩短入睡时间至几分钟 |
| **DLI闭环算法** | I_total = I_natural + I_supplement，全天能量积分精准补光 |
| **ToF测距补偿** | ±1cm精度，执行1/d²平方反比律距离修正 |
| **非对称偏光** | 45°定向投光，防眩光，"见光不见灯" |

## 成本优化逻辑

| 优化项 | 原版 | 优化版 | 节省成本 |
|:---|:---|:---|:---|

### 技术选型建议
- **深度学习框架**：PyTorch Mobile
- **病害检测模型**：ResNet18/34 轻量化
- **种类识别模型**：MobileNetV3
- **部署方案**：云端+边缘混合

### 开发路线建议
**Phase 1：病害检测（优先）**
- 基于PlantDoc数据集训练ResNet模型
- 支持红蜘蛛、白粉病、脱水等5-10种病害
- 目标准确率>85%

**Phase 2：种类识别（其次）**
- 缩小范围至Top 50常见室内植物
- 迁移学习，ImageNet预训练微调
- 目标Top-5准确率>90%

**Phase 3：综合优化**
- 整合双模型，统一识别流水线
- 结合ToF距离信息提升近景识别
