# AeroGrow Pro - 智慧植物养护系统

智慧植物光环境管理系统，为您的植物群落创建最佳光环境。

## 🚀 快速访问
**在线 Demo**: https://ryanx0198.github.io/AeroGrow-Pro


## ✨ 核心功能

| 功能 | 说明 |
|:---|:---|
| 🌿 **植物监测** | DLI、PPFD 实时监测 |
| 💡 **灯光控制** | AR风格空间热力图，可视化PPFD分布 |
| 📊 **数据统计** | 光环境分析，植物生长追踪 |

## 🛠️ 技术栈

- **前端**: 原生 HTML5 + Tailwind CSS + Canvas
- **交互**: 单页应用 (SPA)，无框架依赖
- **可视化**: 自定义 Canvas 热力图渲染

## 📱 使用方式

1. 直接打开 `index.html` 或在浏览器访问在线 Demo
2. 完成植物配置向导
3. 查看实时光环境监测数据
4. 调整灯光参数，查看AR热力图变化
